# 📞 Call Center Performance & Revenue Dashboard

## 📌 Project Overview
An interactive Advanced Excel dashboard designed to track call center operational health, revenue generation, and agent performance. 

*Note: The raw dataset for this project was sourced from Chandoo's analytics resources to practice real-world call center data modeling and visualization techniques.*

## 📊 Key Metrics Analyzed
* **Total Call Volume:** 1,000 calls analyzed across a 5-month period (Jan-May).
* **Financial Impact:** $96,623 in total revenue generated ($96.62 Avg. Amount Per Call).
* **Quality Assurance:** Tracked 307 perfect 5-star calls, maintaining an overall average rating of 3.96.

## 🛠️ Technical Implementation & Design Choices
Rather than just visualizing numbers, this dashboard was built using strict data visualization best practices:

* **Interactive Slicers:** Connected Pivot Tables to a Rep ID (R01-R05) slicer, allowing stakeholders to dynamically filter the entire dashboard to isolate individual agent performance.
* **Intentional Chart Selection:** Deliberately utilized Line Charts to display continuous monthly trends (Jan-May) and Bar Charts for distinct categorical comparisons (Days of the Week).
* **Geographical Matrix:** Cross-referenced revenue by Client Cities (e.g., Cincinnati, Cleveland) and Customer IDs to identify high-value regional accounts.
* **Business Formatting:** Applied proper financial formatting (currency symbols, decimal limiters) to elevate raw data into a clean, executive-ready presentation.
